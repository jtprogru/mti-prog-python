# Лабораторные работы по программированию на Python

Этот репозиторий содержит небольшие самостоятельные скрипты, демонстрирующие базовые возможности Python. Каждый скрипт имеет функцию `main()` и может запускаться непосредственно:

```bash
python <script>.py
```

## Как запустить

1. Установите Python 3.11+ и добавьте его в `PATH`.
2. При желании создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
3. Запустите интересующий скрипт.

## Список скриптов

| Скрипт | Что делает | Как запустить |
|--------|------------|----------------|
| `lab1.py` | Приветствие, демонстрация базовых типов и операций | `python lab1.py` |
| `calculator.py` | Простой калькулятор с арифметикой | `python calculator.py` |
| `anketa.py` | Сбор анкеты пользователя и вывод таблицы | `python anketa.py` |
| `triangle.py` | Вычисление площади и периметра треугольника | `python triangle.py` |
| `multiplication_table.py` | Вывод таблицы умножения | `python multiplication_table.py` |
| `sentence_analysis.py` | Анализ предложения (слова, символы) | `python sentence_analysis.py` |
| `student_grades.py` | Учёт оценок студентов | `python student_grades.py` |
| `inventory.py` | Пример работы с инвентарём | `python inventory.py` |
| `library_system.py` | Симуляция библиотеки (книги, читатели) | `python library_system.py` |
| `main_lab3.py` | Дополнительный пример (описание в файле) | `python main_lab3.py` |
| `string_operations.py` | Операции со строками | `python string_operations.py` |
| `diary_app.py` | Простейшее приложение‑дневник | `python diary_app.py` |

## Примечание

Все скрипты независимы и не используют общую базу данных. При желании можете добавить свои упражнения, следуя тем же конвенциям.
